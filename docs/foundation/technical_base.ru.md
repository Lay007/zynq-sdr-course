# Техническая база

## Назначение
Этот документ даёт общую карту технической базы курса.

## Основные компоненты
- макетная плата и простые схемы
- RTL-SDR свисток
- HDSDR
- GNU Radio
- MATLAB
- Simulink
- утилиты на C++
- SDR-плата Zynq-7020 + AD9363

## Роли
- breadboard-лабораторные учат, что сигнал — это физическое явление
- RTL-SDR + HDSDR — первый инструмент наблюдения сигнала
- MATLAB даёт компактные численные эталоны
- Simulink даёт исполняемые модели сигналов
- GNU Radio даёт живые приёмные тракты и file-based SDR workflow
- C++ даёт повторно используемые парсеры и DSP-утилиты
- Zynq + AD9363 — основная RF-платформа курса
