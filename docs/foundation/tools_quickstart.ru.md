# Быстрый старт по инструментам

## MATLAB
MATLAB используется для генерации сигналов, графиков, сохранения данных и первого оффлайн-анализа записей.

## Simulink
Simulink используется для исполняемых потоковых моделей и параметризованных экспериментов с сигналом.

## HDSDR
HDSDR используется для живого наблюдения, waterfall, скриншотов и записи сигнала.

## GNU Radio
GNU Radio используется для собственных приёмных flowgraph, воспроизведения файлов и SDR-пайплайнов.

## Python
Python используется для загрузчиков, графиков, преобразований и разбора записей.

## C++
C++ используется для эталонных утилит, парсеров и оффлайн DSP-инструментов.

## Ключевой workflow
**запись в HDSDR -> анализ в MATLAB / Simulink / Python / C++ / GNU Radio**
