# Введение в RTL-SDR + HDSDR

RTL-SDR + HDSDR вводится как практический инструмент наблюдения сигнала.

## Основные применения
- живой просмотр спектра
- просмотр waterfall
- перестройка на видимые сигналы
- сохранение скриншотов
- запись сигналов для последующего оффлайн-анализа

На этом этапе глубокая SDR-теория не требуется.
Эта связка используется как доступный прибор наблюдения и записи.
