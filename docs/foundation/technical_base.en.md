# Technical Base

## Purpose
This document provides a high-level map of the technical base used in the course.

## Main components
- breadboard hardware
- RTL-SDR dongle
- HDSDR
- GNU Radio
- MATLAB
- Simulink
- C++ tools
- Zynq-7020 + AD9363 SDR board

## Roles
- breadboard labs teach what a physical signal is
- RTL-SDR + HDSDR is the first signal-viewing instrument
- MATLAB provides compact numerical references
- Simulink provides executable waveform models
- GNU Radio provides live receive chains and file-based SDR pipelines
- C++ provides reusable parsers and DSP utilities
- Zynq + AD9363 provides the main RF transmit/receive platform
