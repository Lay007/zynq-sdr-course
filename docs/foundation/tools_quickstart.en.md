# Tools Quickstart

## MATLAB
Use MATLAB for waveform generation, plotting, saving data, and first offline analysis of recorded signals.

## Simulink
Use Simulink for executable signal-flow models and parameterized waveform experiments.

## HDSDR
Use HDSDR for live viewing, waterfall observation, screenshots, and signal recording.

## GNU Radio
Use GNU Radio for custom receive flowgraphs, file playback, and SDR pipelines.

## Python
Use Python for loaders, plotting, conversion, and recorded-signal analysis.

## C++
Use C++ for reference tools, parsers, and offline DSP utilities.

## Key workflow
**HDSDR recording -> MATLAB / Simulink / Python / C++ / GNU Radio analysis**
