# Мостик от записи к анализу

Живая запись из HDSDR превращается в повторно используемые данные для оффлайн-DSP-анализа.

## Основной маршрут
**живой RF-сигнал -> запись в HDSDR -> оффлайн-анализ**

## Среды анализа
- MATLAB
- Simulink
- Python
- C++
- GNU Radio

Одна и та же запись должна рассматриваться как повторно используемые данные курса.
