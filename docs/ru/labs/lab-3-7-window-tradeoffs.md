# Lab 3.7 — окна FFT и обнаружение слабых сигналов

Эта лабораторная усиливает раздел спектрального анализа в DSP foundation track. Это детерминированная script-driven работа, без notebook-подхода.

## Цель

Показать, что FFT window — это инженерный выбор, а не декоративная настройка графика. Лабораторная сравнивает, как разные окна влияют на видимость слабого тона рядом с сильным тоном.

## Что измеряется

| Свойство окна | Последствие для SDR-измерений |
|---|---|
| Ширина главного лепестка | Разделение тонов и частотное разрешение. |
| Уровень боковых лепестков | Возможность увидеть слабый сигнал рядом с сильным компонентом. |
| Coherent gain | Корректная интерпретация амплитуды. |
| ENBW | Интерпретация noise floor и чувствительности. |

## Инженерные вопросы

1. Почему rectangular window может скрыть слабый сигнал рядом с сильным?
2. Почему снижение боковых лепестков обычно увеличивает ширину главного лепестка?
3. Почему coherent gain нужен для амплитудных измерений?
4. Почему ENBW важен при оценке noise floor?
5. Какое окно лучше выбрать для weak-signal detection и почему?

## Что включить в отчёт

- Оба сгенерированных графика.
- Сравнение минимум трёх типов окон.
- Coherent gain и ENBW.
- Метрику видимости слабого тона.
- Пояснение, как выбор окна влияет на SDR spectrum display и measurement report.
