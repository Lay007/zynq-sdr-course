# Лабораторная 10.2 — Простой аттенюатор

## Цель

Научиться рассчитывать простую цепь ослабления сигнала и понимать, почему аттенюатор обязателен в RF/SDR loopback-экспериментах.

## Что выполняется

В работе студент:

1. выбирает требуемое ослабление;
2. рассчитывает делитель напряжения;
3. переводит отношение напряжений в дБ;
4. анализирует влияние нагрузки;
5. формулирует ограничения учебного аттенюатора.

## Результат

После выполнения работы должны быть получены:

- расчёт номиналов;
- ожидаемое ослабление;
- схема подключения;
- safety conclusion для SDR-стенда.
