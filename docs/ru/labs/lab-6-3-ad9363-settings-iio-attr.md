# Лабораторная 6.3 — Настройки AD9363 и iio_attr

## Цель

Научиться фиксировать параметры AD9363 через IIO/iio_attr так, чтобы RF-эксперимент можно было воспроизвести позднее.

## Что выполняется

В работе студент:

1. находит доступные IIO-устройства;
2. просматривает атрибуты PHY и каналов;
3. задаёт LO frequency, sample rate и RF bandwidth;
4. фиксирует gain mode и hardware gain;
5. оформляет настройки в metadata/checklist.

## Результат

После выполнения работы должны быть получены:

- список используемых IIO-устройств;
- набор команд `iio_attr`;
- таблица настроек AD9363;
- пример metadata для RF-захвата;
- вывод о воспроизводимости эксперимента.

## Что приложить к отчёту

- вывод discovery-команд;
- частоты LO;
- sample rate и RF bandwidth;
- gain settings;
- ссылку на metadata JSON;
- инженерный вывод.
