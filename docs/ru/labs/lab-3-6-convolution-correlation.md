# Lab 3.6 — свёртка и корреляция в SDR

Эта лабораторная разделяет две операции, которые часто смешивают на раннем этапе изучения DSP: свёртку для фильтрации и корреляцию для обнаружения. Это детерминированная script-driven работа, без notebook-подхода.

## Цель

На одном синтетическом SDR-сигнале показать две разные инженерные роли:

| Операция | Роль в SDR | Следствие для FPGA / реализации |
|---|---|---|
| Свёртка | FIR-фильтрация, pulse shaping, отклик канала. | Multiply-accumulate структура, задержка, квантование коэффициентов. |
| Корреляция | Обнаружение преамбулы, оценка задержки, синхронизация. | Скользящий matched filter, ширина аккумулятора, threshold logic. |

2. Почему корреляция является естественной операцией для обнаружения преамбулы?
3. Как шум влияет на корреляционный пик?
4. Какая ширина аккумулятора нужна fixed-point correlator?
5. Как реализовать sliding correlator в FPGA logic?
