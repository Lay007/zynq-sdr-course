# Лабораторная 10.1 — Пассивный RC-фильтр

## Цель

Научиться рассчитывать простой RC-фильтр и объяснять, как он влияет на сигнал перед измерением или SDR-захватом.

## Что выполняется

В работе студент:

1. выбирает тип фильтра;
2. рассчитывает частоту среза;
3. выбирает номиналы R и C;
4. строит схему подключения;
5. оценивает применимость фильтра для SDR-стенда.

## Результат

После выполнения работы должны быть получены:

- расчёт RC-фильтра;
- схема подключения;
- таблица параметров;
- инженерный вывод.
