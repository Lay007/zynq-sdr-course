# Лабораторная 5.3 — RTL-смеситель IQ на основе NCO

## Цель

Реализовать учебный fixed-point IQ-смеситель на Verilog с фазовым аккумулятором, таблицей sin/cos и комплексным умножением.

## Что выполняется

В работе студент:

1. изучает фазовый аккумулятор NCO;
2. анализирует LUT sin/cos в формате Q1.15;
3. запускает RTL-смеситель IQ;
4. генерирует эталонные векторы Python-скриптом;
5. проверяет complex multiply, округление и насыщение.

## Результат

После выполнения работы должны быть получены:

- RTL-модуль `nco_mixer_iq`;
- Python-generated reference vectors;
- self-checking testbench;
- VCD waveform;
- PASS/FAIL лог симуляции.

## Что приложить к отчёту

- описание NCO;
- таблицу fixed-point форматов;
- формулы комплексного умножения;
- результат симуляции;
- вывод о масштабировании к реальному SDR-смесителю.
