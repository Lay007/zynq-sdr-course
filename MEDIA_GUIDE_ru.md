# MEDIA GUIDE

## Что использовать
- фото оборудования: JPG
- скриншоты программ: PNG
- блок-диаграммы и схемы подключения: SVG
- короткие анимации для README/GitHub: GIF
- более качественные видео для сайта: MP4/WebM

## Рекомендуемая структура
```text
images/
├── photos/
├── screenshots/
├── diagrams/
├── schematics/
└── animations/
```

## Правила
- английские имена файлов;
- lowercase;
- слова через underscore;
- фото и скриншоты не смешивать;
- KiCad-исходники хранить отдельно от экспортированных картинок.
