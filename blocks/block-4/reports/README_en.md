# Reports — Block 4

## Purpose
This folder should contain reports, exported plots, parameter tables, and supporting materials for the block “Simulink and Fixed-Point Preparation”.

## Minimum content
- one main report for a laboratory or demonstration task;
- a folder with figures or screenshots;
- a list of model or experiment parameters;
- short conclusions tied to the goals of the block.

## What must be documented
- the problem statement;
- the model, stand, or software environment used;
- experiment parameters;
- observed results;
- engineering conclusions and limits.
