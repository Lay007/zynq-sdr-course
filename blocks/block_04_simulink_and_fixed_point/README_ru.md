# Блок 4. Simulink и fixed-point подготовка

## Назначение
Этот блок подготовлен как двуязычная заготовка для дальнейшего наполнения курса. Он должен продолжить инженерную линию после предыдущих модулей и привести студента к следующему уровню практики.

## Цели блока
- дать системное понимание темы блока;
- связать теорию с экспериментом и инструментами курса;
- подготовить основу для лабораторных и проектных работ.

## Основные темы
1. Построение модели тракта
2. Фиксированная точка
3. Масштабирование и переполнение
4. Проверка модели до HDL
5. Связь с аппаратной реализацией

## Практическая часть
Рекомендуется включить:
- одну демонстрацию;
