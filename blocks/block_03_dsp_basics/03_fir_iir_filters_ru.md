# 03. КИХ- и БИХ-фильтры

## Классификация

| Тип | Полное название | Импульсная характеристика | Обратная связь |
|---|---|---|---|
| КИХ (FIR) | Конечная импульсная характеристика | конечной длины | нет |
| БИХ (IIR) | Бесконечная импульсная характеристика | бесконечной длины | есть |

## КИХ-фильтр

Выходной отсчёт — это взвешенная сумма последних `M+1` входных отсчётов:

```text
y[n] = Σ b[k] · x[n-k],   k = 0 … M
```

Ключевые свойства:

- всегда устойчив (нет полюсов вне единичной окружности);
- линейная фазовая характеристика (при симметричных коэффициентах);
- требует больше операций при той же крутизне спада, чем БИХ.

## БИХ-фильтр

```text
y[n] = Σ b[k]·x[n-k] − Σ a[k]·y[n-k],   k = 0 … M / 1 … N
```

Ключевые свойства:

- нелинейная фазовая характеристика (кроме специальных структур);
- может быть неустойчивым при неправильном проектировании;
- достигает крутого спада при меньшем порядке, чем КИХ.

## Сравнение

| Критерий | КИХ | БИХ |
|---|---|---|
| Устойчивость | гарантирована | требует проверки |
| Фазовая характеристика | линейная (симм.) | нелинейная |
| Реализация в FPGA | умножитель × задержка | сложнее, петли обратной связи |
| Порядок при той же крутизне | высокий | низкий |

## Проектирование КИХ-фильтра

```python
from scipy.signal import firwin, freqz
taps = firwin(numtaps=63, cutoff=0.1)    # cutoff в долях от Nyquist
w, H = freqz(taps, worN=2048)
```

## Проектирование БИХ-фильтра

```python
from scipy.signal import butter, freqz
b, a = butter(N=5, Wn=0.1, btype='low')
w, H = freqz(b, a, worN=2048)
```

## Применение в SDR

- **Низкочастотный КИХ после децимации** — выбор узкой полосы вокруг тона;
- **Компенсационный КИХ после CIC** — выравнивание passband droop;
- **БИХ для быстрой предфильтрации** — предварительное подавление мощных
  мешающих сигналов до децимации.

## Мини-лабораторная

1. Спроектировать КИХ ФНЧ (63 коэффициента, срез 0.1·Nyquist).
2. Применить к IQ-записи с тоном на 50 кГц и помехой на 300 кГц.
3. Построить спектр до и после фильтрации.
4. Измерить ослабление помехи и спад в полосе пропускания.
5. Повторить с БИХ Баттерворта 5-го порядка.
