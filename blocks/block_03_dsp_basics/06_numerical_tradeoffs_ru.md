# 06. Числовые компромиссы в DSP-реализации

## Почему это важно

Переход от математической модели к реальной реализации (Python float64 →
MATLAB fixed-point → FPGA integer) требует понимания численных ограничений
на каждом уровне.

```mermaid
flowchart LR
    FP64["Python / MATLAB float64<br/>~15 значащих цифр<br/>погрешность ≈ 2^-52"]
    FP32["float32<br/>~7 значащих цифр<br/>погрешность ≈ 2^-23"]
    FX["Fixed-point<br/>Q(I.F) формат<br/>конечная разрядная сетка"]
    INT["FPGA integer<br/>N бит, без скрытых переполнений"]

    FP64 --> FP32 --> FX --> INT
```

## Основные источники ошибок

| Источник | Что происходит | Как избежать |
|---|---|---|
| Переполнение | выход за диапазон → зацикливание или насыщение | расширить разрядность или ограничить уровень сигнала |
| Округление коэффициентов | КИХ-коэффициенты теряют точность | использовать достаточную длину слова коэффициентов |
| Накопление ошибок | ошибки складываются по длинным КИХ | увеличить длину аккумулятора |
| Quantization noise | уровень квантования | `σ² ≈ Δ² / 12`, где `Δ = 2^-F` |

## Fixed-point формат Q(I.F)

- `I` — число бит под целую часть (знаковые + незнаковые);
- `F` — число бит под дробную часть;
- разрешение: `LSB = 2^-F`;
- диапазон (знаковый): `[-2^(I-1), 2^(I-1) − 2^-F]`.

Пример: Q3.12 = 3 бита целая + 12 бит дробная = 15 бит + знаковый бит.

## Разрядность в типичных SDR-цепочках

| Блок | Типичная разрядность |
|---|---|
| АЦП IQ (AD9363) | 12 бит |
| Накопление CIC | 12 + N·log2(R) бит |
| Коэффициенты КИХ | 16–18 бит |
| Аккумулятор умножителя | 32–36 бит |
| Выход ЦАП | 12–16 бит |

## Шум квантования

```text
SNR_квантования ≈ 6.02 · B + 1.76 дБ
```

где `B` — число эффективных бит (ENOB). Каждый дополнительный бит добавляет
~6 дБ SNR.

## Практическое правило

При проектировании FPGA-цепочки:

1. Начать с float64 в Python — это "золотой стандарт".
2. Перейти на float32 и проверить, что результат не изменился существенно.
3. Смоделировать fixed-point в MATLAB Fixed-Point Designer.
4. Сравнить выход fixed-point с float64 по SQNR (signal-to-quantization-noise ratio).
5. Написать RTL, убедиться что HDL-симуляция соответствует fixed-point модели.

## Мини-лабораторная

1. Реализовать 31-коэффициентный КИХ ФНЧ в float64.
2. Квантовать коэффициенты до 8 бит → сравнить АЧХ.
3. Квантовать до 16 бит → сравнить АЧХ.
4. Применить оба фильтра к тестовому сигналу и построить разностный спектр.
5. Вычислить SQNR для каждого варианта.
