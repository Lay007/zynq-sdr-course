# Наборы данных и вспомогательные материалы

## Назначение
Эта папка должна содержать таблицы параметров, вспомогательные данные, справочные материалы и маленькие артефакты, которые переиспользуются в лабораторных работах блока для блока 3 «Базовые DSP-операции».

## Что здесь должно лежать
- материалы, напрямую поддерживающие темы блока: FFT и интерпретация спектра; оконные функции и spectral leakage; FIR и IIR-фильтры;
- файлы, которые можно использовать в лабораторной работе, демонстрации или мини-проекте;
- краткие пояснения по запуску, источнику данных и ожидаемому результату.

## Минимум содержательного наполнения
- один воспроизводимый пример по ключевой теме блока;
- один файл или набор артефактов для отчёта;
- одно пояснение, связывающее материалы папки с общей инженерной логикой курса.

## Рекомендуемые имена файлов
- `fft_reference_data.csv`
- `filter_parameter_table.md`
- `multirate_notes.md`

## Критерии качества
- файлы должны быть воспроизводимы и понятны без устных комментариев;
- параметры эксперимента, версии инструментов и ограничения должны быть зафиксированы рядом с материалом;
- материалы должны помогать студенту дойти до одного из результатов блока: набор спектров до и после обработки, параметры фильтров.
