# Блок 3. Базовые DSP-операции

## Назначение
Блок вводит FFT, оконные функции, фильтрацию и базовые операции цифровой обработки, необходимые для следующих модулей курса.

## Почему блок важен
Именно здесь студент получает инструменты для интерпретации спектра, подавления помех и проектирования элементарных цифровых трактов.

## Основные темы
- FFT и интерпретация спектра;
- оконные функции и spectral leakage;
- FIR и IIR-фильтры;
- свёртка и отклик системы;
- децимация и интерполяция;
- численные компромиссы в DSP-реализации.

## Практическая часть
- сравнение окон по уровню лепестков и разрешению;
- проектирование простых фильтров;
- анализ влияния фильтра на IQ-запись;
- сборка мини-цепочки multirate-обработки.

## Инструменты блока
Основной набор инструментов: MATLAB, Python, Simulink, GNU Radio.

## Что должно получиться на выходе
- набор спектров до и после обработки;
- параметры фильтров;
- таблица сравнения окон;
- отчёт с выводами по DSP-компромиссам.

## Рекомендуемый порядок работы
1. проработать FFT и оконные функции.
2. спроектировать базовые фильтры.
3. проверить multirate-операции на моделях и скриптах.
4. оформить сравнение методов обработки.
