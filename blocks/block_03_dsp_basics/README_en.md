# Block 3. Basic DSP Operations

## Purpose
This block is provided as a bilingual scaffold for further course development. It should continue the engineering route from earlier modules and move the student to the next practical level.

## Block objectives
- provide a systematic understanding of the block topic;
- connect theory with experiments and course tools;
- prepare the basis for labs and project work.

## Main topics
1. FFT and spectrum interpretation
2. Window functions
3. Filtering
4. Frequency and level estimation
5. Simple experiments on real data

## Practical part
Recommended inclusions:
- one demonstration;
- one or more laboratory works;
- signal or model recording/analysis;
- a report with plots, screenshots, and conclusions.
