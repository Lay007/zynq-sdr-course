# 03. FFT и частотная ось

## Проблема

FFT возвращает индексы, а не физические частоты.

## Правильная формула

```text
f[k] = (k - N/2) * Fs / N
```

## Связь с SDR

```text
f_absolute = Fc + f_baseband
```

## Ошибки

| Ошибка | Симптом |
|---|---|
| неверный Fs | неправильная шкала |
| отсутствие Fc | нельзя сопоставить с RF |
| нет сдвига FFT | спектр “съехал” |

## Мини-лабораторная

1. Построить FFT без учёта Fs.
2. Построить FFT с правильной осью.
3. Сравнить результат.
