# 06. Метаданные IQ-записи

## Почему это критично

IQ-файл без метаданных почти бесполезен: без `Fs` и `Fc` невозможно правильно интерпретировать частоты.

## Минимальный набор

| Параметр | Значение |
|---|---|
| Fs | частота дискретизации |
| Fc | центральная частота |
| format | тип данных (int16, float32) |
| IQ order | порядок I/Q |
| gain | усиление приёмника |
| duration | длительность записи |

## Типовые ошибки

| Ошибка | Симптом |
|---|---|
| неправильный Fs | неверная шкала частоты |
| отсутствует Fc | невозможно перейти к RF |
| перепутан формат | искажения сигнала |
| нет gain | сложно оценить уровень |

## Чек-лист

Перед анализом всегда фиксируйте:

- Fs
- Fc
- формат
- порядок I/Q
- источник сигнала
