# Короткие анимации изменений параметров

## Назначение
Эта папка хранит короткие GIF или MP4 с изменением параметров, которые трудно объяснить одной статичной картинкой для блока 1 «Введение в SDR, инструменты и первый приём сигнала».

## Что здесь должно лежать
- материалы, напрямую поддерживающие темы блока: архитектура SDR и роль DSP; учебный стенд на базе Zynq7020 + AD9363; внешний приём через RTL-SDR и HDSDR;
- файлы, которые можно использовать в лабораторной работе, демонстрации или мини-проекте;
- краткие пояснения по запуску, источнику данных и ожидаемому результату.

## Минимум содержательного наполнения
- один воспроизводимый пример по ключевой теме блока;
- один файл или набор артефактов для отчёта;
- одно пояснение, связывающее материалы папки с общей инженерной логикой курса.

## Рекомендуемые имена файлов
- `block_1_parameter_sweep.gif`
- `block_1_signal_change.gif`
- `block_1_process_overview.mp4`

## Критерии качества
- файлы должны быть воспроизводимы и понятны без устных комментариев;
- параметры эксперимента, версии инструментов и ограничения должны быть зафиксированы рядом с материалом;
- материалы должны помогать студенту дойти до одного из результатов блока: скриншоты наблюдения сигнала, IQ-запись с метаданными.
