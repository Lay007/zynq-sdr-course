# 02. Подготовка программного окружения

## Цель раздела
Подготовить рабочее программное окружение для первого блока курса и понять назначение каждого инструмента.

## 1. Минимальный набор ПО для старта
1. **RTL-SDR driver**
2. **HDSDR**
3. **MATLAB / Simulink**
4. **Python**
5. **VS Code**

## 2. Расширенный набор ПО
- **GNU Radio**
- **Vivado / Vitis**
- **KiCad**
- **компилятор C/C++**

## 3. Назначение инструментов
- **HDSDR** — быстрое наблюдение спектра и водопада.
- **RTL-SDR** — внешний бюджетный SDR-приёмник.
- **MATLAB / Simulink** — моделирование и анализ.
- **Python** — офлайн-анализ и автоматизация.
- **C/C++** — производительные утилиты.
- **GNU Radio** — визуальная сборка SDR-цепочек.
- **Vivado / Vitis** — путь к реализации на Xilinx.
- **KiCad** — чтение схем и схемотехническая часть.
- **VS Code** — единая рабочая среда проекта.

## 4. Минимальный чек-лист
- RTL-SDR определяется системой.
- HDSDR запускается и видит источник сигнала.
- MATLAB запускается.
- Simulink открывается.
- Python работает из консоли.
- VS Code установлен.
- KiCad запускается.
