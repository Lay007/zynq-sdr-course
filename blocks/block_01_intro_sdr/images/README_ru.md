# Images

- `photos/` — реальные фотографии оборудования и стенда;
- `screenshots/` — снимки окон программ;
- `diagrams/` — блок-схемы и архитектурные рисунки;
- `schematics/` — экспортированные схемы для вставки в Markdown;
- `animations/` — GIF/видео с динамикой сигнала.

Подробные рекомендации см. в `../MEDIA_GUIDE.md`.
