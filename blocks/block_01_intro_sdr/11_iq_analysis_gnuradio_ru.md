# 11. Анализ записанного сигнала в GNU Radio

## Цель раздела
Познакомиться с GNU Radio как со средой визуальной сборки SDR-цепочек и использовать её для анализа записанного IQ-сигнала.

## Минимальная структура flowgraph
- **File Source**
- **Throttle**
- **QT GUI Frequency Sink**
- **QT GUI Time Sink**

## Что важно настроить
- тип данных файла;
- формат комплексного сигнала;
- частоту дискретизации;
- размер FFT;
- длину отображаемого окна.

## Что должен увидеть студент
- устойчивый спектральный пик тестового тона;
- временную структуру сигнала;
- согласование результата с другими средами анализа.
