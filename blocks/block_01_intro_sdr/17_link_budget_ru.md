# 17. Link budget в SDR

## Цель
Оценить уровень сигнала на приёмнике.

## Основная формула

```text
Prx = Ptx - Loss + Gain
```

## Что учитывать

- мощность передатчика;
- потери в кабеле;
- потери в эфире;
- усиление антенны;
- усиление приёмника.

## Практический вывод

Link budget позволяет заранее оценить, будет ли сигнал виден.
