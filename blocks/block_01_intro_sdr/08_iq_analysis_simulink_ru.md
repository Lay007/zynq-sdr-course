# 08. Анализ записанного сигнала в Simulink

## Цель раздела
Научиться использовать Simulink для чтения записанного IQ-сигнала и визуального анализа его временных и спектральных характеристик.

## Что должен сделать студент
- загрузить IQ-файл в модель Simulink;
- представить его как комплексный сигнал;
- вывести временную форму;
- вывести спектр;
- сравнить результат с MATLAB и HDSDR.

## Простейшая модель анализа
1. Источник данных из файла.
2. Формирование каналов I и Q.
3. Объединение в комплексный сигнал.
4. Временной анализ.
5. Спектральный анализ.
6. Отображение результатов.

## Рекомендуемая структура блоков
- File Reader / аналогичный источник
- Data Type Conversion
- Demux / Selector
- Real-Imag to Complex
- Time Scope
- Spectrum Analyzer
