# Схемы и KiCad-проекты

## Назначение
Эта папка предназначена для схем, проектов KiCad и электрических пояснений, которые связывают цифровую часть курса с реальными соединениями и вспомогательными узлами для блока 2 «Сигналы, спектр, дискретизация и I/Q».

## Что здесь должно лежать
- материалы, напрямую поддерживающие темы блока: непрерывные и дискретные сигналы; частотная область и полоса сигнала; частота дискретизации и aliasing;
- файлы, которые можно использовать в лабораторной работе, демонстрации или мини-проекте;
- краткие пояснения по запуску, источнику данных и ожидаемому результату.

## Минимум содержательного наполнения
- один воспроизводимый пример по ключевой теме блока;
- один файл или набор артефактов для отчёта;
- одно пояснение, связывающее материалы папки с общей инженерной логикой курса.

## Рекомендуемые имена файлов
- `sampling_adapter.kicad_pro`
- `aliasing_support_board.kicad_sch`
- `iq_wiring_notes.md`

## Критерии качества
- файлы должны быть воспроизводимы и понятны без устных комментариев;
- параметры эксперимента, версии инструментов и ограничения должны быть зафиксированы рядом с материалом;
- материалы должны помогать студенту дойти до одного из результатов блока: графики времени и спектра, таблица параметров дискретизации.
