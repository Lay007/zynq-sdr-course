# Lab 01 — From IQ Model to Real RF Tone

Goal:
1. generate a complex IQ tone in MATLAB
2. build the same tone in Simulink
3. prepare and transmit it on the Zynq + AD9363 board
4. receive it with RTL-SDR
5. observe it in HDSDR

## Recording extension
- receive the transmitted tone in HDSDR
- record the signal
- analyze the recording offline in MATLAB / Simulink / Python / C++ / GNU Radio
