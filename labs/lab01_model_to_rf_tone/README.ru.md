# Лабораторная 01 — От модели IQ-сигнала до реального RF-тона

Цель:
1. сгенерировать комплексный IQ-тон в MATLAB
2. собрать тот же тон в Simulink
3. подготовить и передать его с платы Zynq + AD9363
4. принять его на RTL-SDR
5. увидеть его в HDSDR

## Расширение через запись
- принять переданный тон в HDSDR
- записать сигнал
- разобрать запись оффлайн в MATLAB / Simulink / Python / C++ / GNU Radio
